\subsection{Detailed Description}
The \hyperlink{class_qxt_confirmation_message}{Qxt\-Confirmation\-Message} class provides a confirmation message. 

Qxt\-Gui \hyperlink{class_qxt_confirmation_message}{Qxt\-Confirmation\-Message} is a confirmation message with checkable  \{\char`\"{}\-Do not show again.\char`\"{}\} option. A checked and accepted confirmation message is no more shown until reseted.

Shows the confirmation message if necessary. The confirmation message is not shown in case  \{\char`\"{}\-Do not show again.\char`\"{}\} has been checked while the same confirmation message was earlierly accepted.

A confirmation message is identified by the combination of title, Q\-Message\-Box\-::text and optional Q\-Message\-Box\-::informative\-Text.

A clicked button with role Q\-Dialog\-Button\-Box\-::\-Accept\-Role or Q\-Dialog\-Button\-Box\-::\-Yes\-Role is considered as \char`\"{}accepted\char`\"{}.
