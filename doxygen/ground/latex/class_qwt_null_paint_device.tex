\subsection{Detailed Description}
A null paint device doing nothing. 

Sometimes important layout/rendering geometries are not available or changable from the public Qt class interface. ( f.\-e hidden in the style implementation ).

\hyperlink{class_qwt_null_paint_device}{Qwt\-Null\-Paint\-Device} can be used to manipulate or filter out these informations by analyzing the stream of paint primitives.

F.\-e. \hyperlink{class_qwt_null_paint_device}{Qwt\-Null\-Paint\-Device} is used by \hyperlink{class_qwt_plot_canvas}{Qwt\-Plot\-Canvas} to identify styled backgrounds with rounded corners.
