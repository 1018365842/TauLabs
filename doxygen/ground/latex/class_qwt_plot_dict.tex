\subsection{Detailed Description}
A dictionary for plot items. 

\hyperlink{class_qwt_plot_dict}{Qwt\-Plot\-Dict} organizes plot items in increasing z-\/order. If \hyperlink{class_qwt_plot_dict_a10dfae8b454fa4258b0372366013507a}{auto\-Delete()} is enabled, all attached items will be deleted in the destructor of the dictionary.
