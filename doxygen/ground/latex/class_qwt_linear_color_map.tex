\subsection{Detailed Description}
\hyperlink{class_qwt_linear_color_map}{Qwt\-Linear\-Color\-Map} builds a color map from color stops. 

A color stop is a color at a specific position. The valid range for the positions is \mbox{[}0.\-0, 1.\-0\mbox{]}. When mapping a value into a color it is translated into this interval according to \hyperlink{class_qwt_linear_color_map_a9ec309df6ec88472a63b14ac2c692c96}{mode()}.
