The \hyperlink{class_qxt_fifo}{Qxt\-Fifo} class provides a simple loopback Q\-I\-O\-Device. 

Qxt\-Core

read and write to the same object emits a ready\-Read Signal. useful for loopback tests where Q\-Buffer does not work.
