\subsubsection*{template$<$class T$>$class aligned\-\_\-allocator$<$ T $>$}

stl compatible allocator to use with with 16 byte aligned types 

Example\-: 
\begin{DoxyCode}
\textcolor{comment}{// Matrix4f requires 16 bytes alignment:}

      my\_map\_mat4;
