\begin{DoxyReturn}{Returns}
the Euler-\/angles of the rotation matrix {\ttfamily $\ast$this} using the convention defined by the triplet ({\itshape a0},{\itshape a1},{\itshape a2})
\end{DoxyReturn}
Each of the three parameters {\itshape a0},{\itshape a1},{\itshape a2} represents the respective rotation axis as an integer in \{0,1,2\}. For instance, in\-: 
\begin{DoxyCode}
Vector3f ea = \hyperlink{uavobjecttemplate_8m_a16a51e808b16c46bbfd36da2e37cd123}{mat}.eulerAngles(2, 0, 2); 
\end{DoxyCode}
 \char`\"{}2\char`\"{} represents the z axis and \char`\"{}0\char`\"{} the x axis, etc. The returned angles are such that we have the following equality\-: 
\begin{DoxyCode}
\hyperlink{uavobjecttemplate_8m_a16a51e808b16c46bbfd36da2e37cd123}{mat} == \hyperlink{group___geometry___module_ga811d6fdab2002723bc7a72f055ce8c1d}{AngleAxisf}(ea[0], Vector3f::UnitZ())
     * \hyperlink{group___geometry___module_ga811d6fdab2002723bc7a72f055ce8c1d}{AngleAxisf}(ea[1], Vector3f::UnitX())
     * \hyperlink{group___geometry___module_ga811d6fdab2002723bc7a72f055ce8c1d}{AngleAxisf}(ea[2], Vector3f::UnitZ()); 
\end{DoxyCode}
 This corresponds to the right-\/multiply conventions (with right hand side frames).
