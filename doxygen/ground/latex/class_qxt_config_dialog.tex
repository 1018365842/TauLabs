\subsection{Detailed Description}
The \hyperlink{class_qxt_config_dialog}{Qxt\-Config\-Dialog} class provides a configuration dialog. 

Qxt\-Gui \hyperlink{class_qxt_config_dialog}{Qxt\-Config\-Dialog} provides a convenient interface for building common configuration dialogs. \hyperlink{class_qxt_config_dialog}{Qxt\-Config\-Dialog} consists of a list of icons and a stack of pages.

Example usage\-: 
\begin{DoxyCode}
\hyperlink{class_qxt_config_dialog}{QxtConfigDialog} dialog;
dialog.addPage(\textcolor{keyword}{new} ConfigurationPage(&dialog), QIcon(\textcolor{stringliteral}{":/images/config.png"}), tr(\textcolor{stringliteral}{"Configuration"}));
dialog.addPage(\textcolor{keyword}{new} UpdatePage(&dialog), QIcon(\textcolor{stringliteral}{":/images/update.png"}), tr(\textcolor{stringliteral}{"Update"}));
dialog.addPage(\textcolor{keyword}{new} QueryPage(&dialog), QIcon(\textcolor{stringliteral}{":/images/query.png"}), tr(\textcolor{stringliteral}{"Query"}));
dialog.exec();
\end{DoxyCode}
