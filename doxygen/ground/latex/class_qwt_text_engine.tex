\subsection{Detailed Description}
Abstract base class for rendering text strings. 

A text engine is responsible for rendering texts for a specific text format. They are used by \hyperlink{class_qwt_text}{Qwt\-Text} to render a text.

\hyperlink{class_qwt_plain_text_engine}{Qwt\-Plain\-Text\-Engine} and \hyperlink{class_qwt_rich_text_engine}{Qwt\-Rich\-Text\-Engine} are part of the \hyperlink{namespace_qwt}{Qwt} library. The implementation of Qwt\-Math\-M\-L\-Text\-Engine uses code from the Qt solution package. Because of license implications it is built into a separate library.
