\subsection{Detailed Description}
A state machine for polygon selections. 

Pressing \hyperlink{class_qwt_event_pattern_af48dc808ac5e125eeed41a4c0285d1eaa5875976eba018ca30eb30ba92986d270}{Qwt\-Event\-Pattern\-::\-Mouse\-Select1} or \hyperlink{class_qwt_event_pattern_a8fb57ceb9982d5583a1bf568e37d3007a32afc68b8e31079c00666a251d0b9c25}{Qwt\-Event\-Pattern\-::\-Key\-Select1} starts the selection and selects the first point, or appends a point. Pressing \hyperlink{class_qwt_event_pattern_af48dc808ac5e125eeed41a4c0285d1eaac4185682db6cfd9c59955eb4cf052fd1}{Qwt\-Event\-Pattern\-::\-Mouse\-Select2} or \hyperlink{class_qwt_event_pattern_a8fb57ceb9982d5583a1bf568e37d3007acfd4f64d5b3b29214e51afcc56ea7eaf}{Qwt\-Event\-Pattern\-::\-Key\-Select2} appends the last point and terminates the selection.
