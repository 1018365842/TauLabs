\subsection{Detailed Description}
\subsubsection*{template$<$typename Matrix\-Type, typename Member\-Op, int Direction$>$class Partial\-Redux\-Expr$<$ Matrix\-Type, Member\-Op, Direction $>$}

Generic expression of a partially reduxed matrix. 


\begin{DoxyParams}{Parameters}
{\em Matrix\-Type} & the type of the matrix we are applying the redux operation \\
\hline
{\em Member\-Op} & type of the member functor \\
\hline
{\em Direction} & indicates the direction of the redux (Vertical or Horizontal)\\
\hline
\end{DoxyParams}
This class represents an expression of a partial redux operator of a matrix. It is the return type of \hyperlink{class_partial_redux}{Partial\-Redux} functions, and most of the time this is the only way it is used.
