\subsection{Detailed Description}
The \hyperlink{class_qxt_letter_box_widget}{Qxt\-Letter\-Box\-Widget} widget is a letter box widget. 

Qxt\-Gui \hyperlink{class_qxt_letter_box_widget}{Qxt\-Letter\-Box\-Widget} preserves the aspect ratio of its content widget.

Use Q\-Widget\-::size\-Increment to define the aspect ratio\-: 
\begin{DoxyCode}
\hyperlink{class_qxt_letter_box_widget}{QxtLetterBoxWidget} letterBox;
widget->setSizeIncrement(16, 9);
letterBox.setWidget(widget);
letterBox.show();
\end{DoxyCode}
