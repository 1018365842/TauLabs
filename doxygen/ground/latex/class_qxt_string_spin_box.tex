\subsection{Detailed Description}
The \hyperlink{class_qxt_string_spin_box}{Qxt\-String\-Spin\-Box} widget is a spin box with string items. 

Qxt\-Gui \hyperlink{class_qxt_string_spin_box}{Qxt\-String\-Spin\-Box} is spin box that takes strings. \hyperlink{class_qxt_string_spin_box}{Qxt\-String\-Spin\-Box} allows the user to choose a value by clicking the up and down buttons or by pressing Up or Down on the keyboard to increase or decrease the value currently displayed. The user can also type the value in manually.
