\subsection{Detailed Description}
\subsubsection*{template$<$typename Binary\-Op, typename Lhs, typename Rhs$>$class Cwise\-Binary\-Op$<$ Binary\-Op, Lhs, Rhs $>$}

Generic expression of a coefficient-\/wise operator between two matrices or vectors. 


\begin{DoxyParams}{Parameters}
{\em Binary\-Op} & template functor implementing the operator \\
\hline
{\em Lhs} & the type of the left-\/hand side \\
\hline
{\em Rhs} & the type of the right-\/hand side\\
\hline
\end{DoxyParams}
This class represents an expression of a generic binary operator of two matrices or vectors. It is the return type of the operator+, operator-\/, and the \hyperlink{class_cwise}{Cwise} methods, and most of the time this is the only way it is used.

However, if you want to write a function returning such an expression, you will need to use this class.
