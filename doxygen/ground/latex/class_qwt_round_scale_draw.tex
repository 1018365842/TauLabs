\subsection{Detailed Description}
A class for drawing round scales. 

\hyperlink{class_qwt_round_scale_draw}{Qwt\-Round\-Scale\-Draw} can be used to draw round scales. The circle segment can be adjusted by \hyperlink{class_qwt_round_scale_draw_a5d85678fdb9fbb4d622425aab9ecc681}{Qwt\-Round\-Scale\-Draw\-::set\-Angle\-Range()}. The geometry of the scale can be specified with \hyperlink{class_qwt_round_scale_draw_af7e08b85826c5e1e5b1762fa07830107}{Qwt\-Round\-Scale\-Draw\-::move\-Center()} and \hyperlink{class_qwt_round_scale_draw_a069d658989db2ad96f26b79a298f7d10}{Qwt\-Round\-Scale\-Draw\-::set\-Radius()}.

After a scale division has been specified as a \hyperlink{class_qwt_scale_div}{Qwt\-Scale\-Div} object using \hyperlink{class_qwt_abstract_scale_draw_a4399aac94a294f5ed6c52114dde00d2f}{Qwt\-Abstract\-Scale\-Draw\-::set\-Scale\-Div(const Qwt\-Scale\-Div \&s)}, the scale can be drawn with the \hyperlink{class_qwt_abstract_scale_draw_aca3c3a7499112f52616d0ee8518fa5a6}{Qwt\-Abstract\-Scale\-Draw\-::draw()} member.

Adjust the baseline circle segment for round scales. 

The baseline will be drawn from min(angle1,angle2) to max(angle1, angle2). The default setting is \mbox{[} -\/135, 135 \mbox{]}. An angle of 0 degrees corresponds to the 12 o'clock position, and positive angles count in a clockwise direction.

\end{DoxyParams}
\begin{DoxyWarning}{Warning}

\begin{DoxyItemize}
\item The angle range is limited to \mbox{[}-\/360, 360\mbox{]} degrees. Angles exceeding this range will be clipped. 
\item For angles more than 359 degrees above or below min(angle1, angle2), scale marks will not be drawn. 
\item If you need a counterclockwise scale, use Qwt\-Scale\-Div\-::set\-Range 
\end{DoxyItemize}
