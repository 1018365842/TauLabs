\subsection{Detailed Description}
\subsubsection*{template$<$typename T$>$class Qwt\-Series\-Data$<$ T $>$}

Abstract interface for iterating over samples. 

\hyperlink{namespace_qwt}{Qwt} offers several implementations of the \hyperlink{class_qwt_series_data}{Qwt\-Series\-Data} A\-P\-I, but in situations, where data of an application specific format needs to be displayed, without having to copy it, it is recommended to implement an individual data access.
