\subsection{Detailed Description}
\subsubsection*{template$<$class K\-E\-Y, class V\-A\-L$>$class Qxt\-Bdb\-Hash$<$ K\-E\-Y, V\-A\-L $>$}

The \hyperlink{class_qxt_bdb_hash}{Qxt\-Bdb\-Hash} class is a template class that provides key/value access to a berkeley db file. 

Qxt\-Berkeley Both value and key must be registered with the qt meta system. You may not touch the file while a \hyperlink{class_qxt_bdb_hash}{Qxt\-Bdb\-Hash} instance is running on it.

examples usage\-: 
\begin{DoxyCode}
\hyperlink{class_qxt_bdb_hash}{QxtBdbHash<qreal,QStringList>} db(\textcolor{stringliteral}{"test.db"});
db.insert(344.4,QStringList<<\textcolor{stringliteral}{"HI"}<<\textcolor{stringliteral}{":)"});
qDebug()<<db[344.4];
\end{DoxyCode}


There is an extensive example in /examples/berkeley/adressbook.

All functions of this class are thread safe. Calling \hyperlink{class_qxt_bdb_hash_ad5cbf60bd04c339d2904ba6cde243a3d}{open()} multiple times is undefined. An iterator may only be used from one thread at once, but you can have multiple iterators.
