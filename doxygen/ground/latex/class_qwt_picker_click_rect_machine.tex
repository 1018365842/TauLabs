\subsection{Detailed Description}
A state machine for rectangle selections. 

Pressing \hyperlink{class_qwt_event_pattern_af48dc808ac5e125eeed41a4c0285d1eaa5875976eba018ca30eb30ba92986d270}{Qwt\-Event\-Pattern\-::\-Mouse\-Select1} starts the selection, releasing it selects the first point. Pressing it again selects the second point and terminates the selection. Pressing \hyperlink{class_qwt_event_pattern_a8fb57ceb9982d5583a1bf568e37d3007a32afc68b8e31079c00666a251d0b9c25}{Qwt\-Event\-Pattern\-::\-Key\-Select1} also starts the selection, a second press selects the first point. A third one selects the second point and terminates the selection.
