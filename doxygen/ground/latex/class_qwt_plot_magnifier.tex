\subsection{Detailed Description}
\hyperlink{class_qwt_plot_magnifier}{Qwt\-Plot\-Magnifier} provides zooming, by magnifying in steps. 

Using \hyperlink{class_qwt_plot_magnifier}{Qwt\-Plot\-Magnifier} a plot can be zoomed in/out in steps using keys, the mouse wheel or moving a mouse button in vertical direction.

Together with \hyperlink{class_qwt_plot_zoomer}{Qwt\-Plot\-Zoomer} and \hyperlink{class_qwt_plot_panner}{Qwt\-Plot\-Panner} it is possible to implement individual and powerful navigation of the plot canvas.
