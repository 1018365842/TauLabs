Pseudo expression providing matrix output with given format. 


\begin{DoxyParams}{Parameters}
{\em Expression\-Type} & the type of the object on which I\-O stream operations are performed\\

This class represents an expression with stream operators controlled by a given \hyperlink{struct_i_o_format}{I\-O\-Format}. It is the return type of \hyperlink{class_matrix_base_a450797ce64a44162dfd4a0cbb4ea6f16}{Matrix\-Base\-::format()} and most of the time this is the only way it is used.
