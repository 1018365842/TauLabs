\subsection{Detailed Description}
A class which controls a value within an interval. 

This class is useful as a base class or a member for sliders. It represents an interval of type double within which a value can be moved. The value can be either an arbitrary point inside the interval (see \hyperlink{class_qwt_double_range_af6a3aa74c970a2bef29f0c1af99720e8}{Qwt\-Double\-Range\-::set\-Value}), or it can be fitted into a step raster (see \hyperlink{class_qwt_double_range_a63bc8ea5e7ecd776b7d32d21db1ea668}{Qwt\-Double\-Range\-::fit\-Value} and \hyperlink{class_qwt_double_range_a299c35c72e9b69f5a0e3dcd2b7da775b}{Qwt\-Double\-Range\-::inc\-Value}).

As a special case, a \hyperlink{class_qwt_double_range}{Qwt\-Double\-Range} can be periodic, which means that a value outside the interval will be mapped to a value inside the interval when \hyperlink{class_qwt_double_range_af6a3aa74c970a2bef29f0c1af99720e8}{Qwt\-Double\-Range\-::set\-Value()}, \hyperlink{class_qwt_double_range_a63bc8ea5e7ecd776b7d32d21db1ea668}{Qwt\-Double\-Range\-::fit\-Value()}, \hyperlink{class_qwt_double_range_a299c35c72e9b69f5a0e3dcd2b7da775b}{Qwt\-Double\-Range\-::inc\-Value()} or \hyperlink{class_qwt_double_range_a19b85337d4ab0b5f0579b02e4fb29a8f}{Qwt\-Double\-Range\-::inc\-Pages()} are called.
