Creates a binding to the provided signal, slot, or Q\-\_\-\-I\-N\-V\-O\-K\-A\-B\-L\-E method using the provided parameter list. The type of each argument is deduced from the type of the Q\-Variant. This function cannot bind positional arguments; see the overload using Q\-Generic\-Argument.

If the provided Q\-Object does not implement the requested method, or if the argument list is incompatible with the method's function signature, this function returns N\-U\-L\-L.

The returned Qxt\-Bound\-Function is created as a child of the receiver. Changing the parent will result in undefined behavior.
