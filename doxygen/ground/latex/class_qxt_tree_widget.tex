\subsection{Detailed Description}
The \hyperlink{class_qxt_tree_widget}{Qxt\-Tree\-Widget} class is an extended Q\-Tree\-Widget with additional signals. 

Qxt\-Gui \hyperlink{class_qxt_tree_widget}{Qxt\-Tree\-Widget} offers an optional top level item decoration and a few most commonly requested signals.

Top level items are decorated according to this property. The default value is Qxt\-::\-No\-Decoration.

\{Note\-:\} Setting the property to anything else than Qxt\-::\-No\-Decoration hides the header and sets Q\-Tree\-View\-::root\-Is\-Decorated to {\ttfamily false} (to avoid multiple branch indicators).
