\subsection{Detailed Description}
The \hyperlink{class_qxt_abstract_http_connector}{Qxt\-Abstract\-Http\-Connector} class is a base class for defining H\-T\-T\-P-\/based protocols for use with \hyperlink{class_qxt_http_session_manager}{Qxt\-Http\-Session\-Manager}. 

Qxt\-Web

\hyperlink{class_qxt_http_session_manager}{Qxt\-Http\-Session\-Manager} does the work of managing sessions and state for the otherwise stateless H\-T\-T\-P protocol, but it relies on \hyperlink{class_qxt_abstract_http_connector}{Qxt\-Abstract\-Http\-Connector} subclasses to implement the protocol used to communicate with the web server.

Subclasses are responsible for accepting new connections (by implementing \hyperlink{class_qxt_abstract_http_connector_acda112d498fcb274bd843aca5deea365}{listen(const Q\-Host\-Address\&, quint16)} and invoking \hyperlink{class_qxt_abstract_http_connector_a0aec454f20271be6c9c087b2e12fbb2c}{add\-Connection(\-Q\-I\-O\-Device$\ast$)}), for informing the session manager when request headers are available (by implementing \hyperlink{class_qxt_abstract_http_connector_a49d287a799d4f07fb78803adc76032c4}{can\-Parse\-Request(const Q\-Byte\-Array\&)}), for parsing the request headers (by implementing \hyperlink{class_qxt_abstract_http_connector_aefaa71fed3832100acf8bbf432968d4f}{parse\-Request(\-Q\-Byte\-Array\&)}), and for writing response headers (by implementing \hyperlink{class_qxt_abstract_http_connector_aecd487f983357c571dc287a64cf3422c}{write\-Headers(\-Q\-I\-O\-Device$\ast$, const Q\-Http\-Response\-Header\&)}).
