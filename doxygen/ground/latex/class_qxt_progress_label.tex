\subsection{Detailed Description}
The \hyperlink{class_qxt_progress_label}{Qxt\-Progress\-Label} widget is a label showing progress related time values. 

Qxt\-Gui \hyperlink{class_qxt_progress_label}{Qxt\-Progress\-Label} is a label widget able to show elapsed and remaining time of a progress. Usage is as simple as connecting signal Q\-Progress\-Bar\-::value\-Changed() to slot \hyperlink{class_qxt_progress_label_a8d96e47266a92c24eddef75ca8646705}{Qxt\-Progress\-Label\-::set\-Value()}.

Example usage\-: 
\begin{DoxyCode}
QProgressBar* bar = \textcolor{keyword}{new} QProgressBar(\textcolor{keyword}{this});
\hyperlink{class_qxt_progress_label}{QxtProgressLabel}* label = \textcolor{keyword}{new} \hyperlink{class_qxt_progress_label}{QxtProgressLabel}(\textcolor{keyword}{this});
\hyperlink{namespace_qxt_meta_object_a0672fb5917d321762dfd192f229469d5}{connect}(bar, SIGNAL(valueChanged(\textcolor{keywordtype}{int})), label, SLOT(\hyperlink{class_qxt_progress_label_a8d96e47266a92c24eddef75ca8646705}{setValue}(\textcolor{keywordtype}{int})));
\end{DoxyCode}
