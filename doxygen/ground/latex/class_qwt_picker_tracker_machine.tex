\subsection{Detailed Description}
A state machine for indicating mouse movements. 

\hyperlink{class_qwt_picker_tracker_machine}{Qwt\-Picker\-Tracker\-Machine} supports displaying information corresponding to mouse movements, but is not intended for selecting anything. Begin/\-End are related to Enter/\-Leave events.
