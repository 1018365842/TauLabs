\subsection{Detailed Description}
The \hyperlink{class_qxt_label}{Qxt\-Label} widget is a label which is able to show elided and rotated plain text. 

Qxt\-Gui \hyperlink{class_qxt_label}{Qxt\-Label} is a label which is able to show elided and rotated plain text. In addition, \hyperlink{class_qxt_label}{Qxt\-Label} provides a signal for clicking.
