\subsection{Detailed Description}
\hyperlink{class_qwt_system_clock}{Qwt\-System\-Clock} provides high resolution clock time functions. 

Sometimes the resolution offered by Q\-Time ( millisecond ) is not accurate enough for implementing time measurements ( f.\-e. sampling ). \hyperlink{class_qwt_system_clock}{Qwt\-System\-Clock} offers a subset of the Q\-Time functionality using higher resolution timers ( if possible ).

Precision and time intervals are multiples of milliseconds (ms).

\begin{DoxyNote}{Note}
The implementation uses high-\/resolution performance counter on Windows, mach\-\_\-absolute\-\_\-time() on the Mac or P\-O\-S\-I\-X timers on other systems. If none is available it falls back on Q\-Timer. 
\end{DoxyNote}
