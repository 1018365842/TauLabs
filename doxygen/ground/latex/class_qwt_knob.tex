\subsection{Detailed Description}
The Knob \hyperlink{class_widget}{Widget}. 

The \hyperlink{class_qwt_knob}{Qwt\-Knob} widget imitates look and behaviour of a volume knob on a radio. It contains a scale around the knob which is set up automatically or can be configured manually (see \hyperlink{class_qwt_abstract_scale}{Qwt\-Abstract\-Scale}). Automatic scrolling is enabled when the user presses a mouse button on the scale. For a description of signals, slots and other members, see \hyperlink{class_qwt_abstract_slider}{Qwt\-Abstract\-Slider}.
