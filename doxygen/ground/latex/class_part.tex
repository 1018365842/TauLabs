\subsection{Detailed Description}
\subsubsection*{template$<$typename Matrix\-Type, unsigned int Mode$>$class Part$<$ Matrix\-Type, Mode $>$}

Expression of a triangular matrix extracted from a given matrix. 


\begin{DoxyParams}{Parameters}
{\em Matrix\-Type} & the type of the object in which we are taking the triangular part \\
\hline
{\em Mode} & the kind of triangular matrix expression to construct. Can be Upper\-Triangular, Strictly\-Upper\-Triangular, Unit\-Upper\-Triangular, Lower\-Triangular, Strictly\-Lower\-Triangular, Unit\-Lower\-Triangular. This is in fact a bit field; it must have either Upper\-Triangular\-Bit or Lower\-Triangular\-Bit, and additionnaly it may have either Zero\-Diag\-Bit or Unit\-Diag\-Bit.\\
\hline
\end{DoxyParams}
This class represents an expression of the upper or lower triangular part of a square matrix, possibly with a further assumption on the diagonal. It is the return type of \hyperlink{class_matrix_base_a2d43a96022ff5bee4b6cf15ce3a63e4a}{Matrix\-Base\-::part()} and most of the time this is the only way it is used.
