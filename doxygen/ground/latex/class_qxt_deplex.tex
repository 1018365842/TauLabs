\subsection{Detailed Description}
The \hyperlink{class_qxt_deplex}{Qxt\-Deplex} class puts a standard Q\-I\-O\-Device into a \hyperlink{class_qxt_pipe}{Qxt\-Pipe} chain. 

Qxt\-Core

implements a normal \hyperlink{class_qxt_pipe}{Qxt\-Pipe} with the exception that r/w operations are delegated to the device set by \hyperlink{class_qxt_deplex_a179252f9e59c44ffeb211df015b48c8b}{set\-Device()}
