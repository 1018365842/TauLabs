\subsection{Detailed Description}
The \hyperlink{class_qxt_stars}{Qxt\-Stars} widget is a stars assessment widget. 

Qxt\-Gui \hyperlink{class_qxt_stars}{Qxt\-Stars} is an optionally interactive stars assessment widget.

By default, orientation is Qt\-::\-Horizontal and range is from {\ttfamily 0} to {\ttfamily 5}.

The stars are rendered accoring to palette with the following color roles\-:    Color\-Role  Element   Q\-Palette\-::\-Text  outlines   Q\-Palette\-::\-Base  unselected stars   Q\-Palette\-::\-Highlight  selected stars
