\subsection{Detailed Description}
\hyperlink{class_qwt_plot_picker}{Qwt\-Plot\-Picker} provides selections on a plot canvas. 

\hyperlink{class_qwt_plot_picker}{Qwt\-Plot\-Picker} is a \hyperlink{class_qwt_picker}{Qwt\-Picker} tailored for selections on a plot canvas. It is set to a x-\/\-Axis and y-\/\-Axis and translates all pixel coordinates into this coodinate system.
