\subsection{Detailed Description}
The \hyperlink{class_qxt_country_model}{Qxt\-Country\-Model} class is a subclass of Q\-Abstract\-Table\-Model to display country data. 

Qxt\-Gui \hyperlink{class_qxt_country_model}{Qxt\-Country\-Model} is a specialized Q\-Abstract\-Table\-Model to display countries. The countries are taken from Q\-Locale\-::\-Country.
