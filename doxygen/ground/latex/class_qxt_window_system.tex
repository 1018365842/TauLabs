\subsection{Detailed Description}
The \hyperlink{class_qxt_window_system}{Qxt\-Window\-System} class provides means for accessing native windows. 

Qxt\-Gui  \{Note\-:\} The underlying window system might or might not allow one to alter states of windows belonging to other processes.

\begin{DoxyWarning}{Warning}
\hyperlink{class_qxt_window_system}{Qxt\-Window\-System} is portable in principle, but be careful while using it since you are probably about to do something non-\/portable.
\end{DoxyWarning}
