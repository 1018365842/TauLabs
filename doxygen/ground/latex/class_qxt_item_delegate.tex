\subsection{Detailed Description}
The \hyperlink{class_qxt_item_delegate}{Qxt\-Item\-Delegate} class is an extended Q\-Item\-Delegate. 

Qxt\-Gui \hyperlink{class_qxt_item_delegate}{Qxt\-Item\-Delegate} provides signals for starting and finishing of editing and an optional decoration of top level items in a Q\-Tree\-View.

\hyperlink{class_qxt_item_delegate}{Qxt\-Item\-Delegate} can also draw a progress bar for items providing appropriate progress data. Just like Q\-Progress\-Bar, \hyperlink{class_qxt_item_delegate}{Qxt\-Item\-Delegate} can show a busy indicator. If minimum and maximum both are set to {\ttfamily 0}, a busy indicator is shown instead of a percentage of steps.
