\subsection{Detailed Description}
The \hyperlink{class_qxt_global_shortcut}{Qxt\-Global\-Shortcut} class provides a global shortcut aka \char`\"{}hotkey\char`\"{}. 

Qxt\-Gui A global shortcut triggers even if the application is not active. This makes it easy to implement applications that react to certain shortcuts still if some other application is active or if the application is for example minimized to the system tray.

Example usage\-: 
\begin{DoxyCode}
\hyperlink{class_qxt_global_shortcut}{QxtGlobalShortcut}* shortcut = \textcolor{keyword}{new} \hyperlink{class_qxt_global_shortcut}{QxtGlobalShortcut}(window);
\hyperlink{namespace_qxt_meta_object_a0672fb5917d321762dfd192f229469d5}{connect}(shortcut, SIGNAL(\hyperlink{class_qxt_global_shortcut_ae5048554f63055a79bc708e231cb8818}{activated}()), window, SLOT(toggleVisibility()));
shortcut->setShortcut(QKeySequence(\textcolor{stringliteral}{"Ctrl+Shift+F12"}));
\end{DoxyCode}


\{Note\-:\} Since Qxt 0.\-6 \hyperlink{class_qxt_global_shortcut}{Qxt\-Global\-Shortcut} no more requires \hyperlink{class_qxt_application}{Qxt\-Application}.
