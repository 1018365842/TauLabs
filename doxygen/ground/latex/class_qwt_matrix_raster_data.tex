\subsection{Detailed Description}
A class representing a matrix of values as raster data. 

\hyperlink{class_qwt_matrix_raster_data}{Qwt\-Matrix\-Raster\-Data} implements an interface for a matrix of equidistant values, that can be used by a \hyperlink{class_qwt_plot_raster_item}{Qwt\-Plot\-Raster\-Item}. It implements a couple of resampling algorithms, to provide values for positions, that or not on the value matrix.

Assign the bounding interval for an axis. 

Setting the bounding intervals for the X/\-Y axis is mandatory to define the positions for the values of the value matrix. The interval in Z direction defines the possible range for the values in the matrix, what is f.\-e used by \hyperlink{class_qwt_plot_spectrogram}{Qwt\-Plot\-Spectrogram} to map values to colors. The Z-\/interval might be the bounding interval of the values in the matrix, but usually it isn't. ( f.\-e a interval of 0.\-0-\/100.\-0 for values in percentage )
