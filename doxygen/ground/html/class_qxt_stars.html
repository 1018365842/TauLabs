Detailed Description

The QxtStars widget is a stars assessment widget.

QxtGui QxtStars is an optionally interactive stars assessment widget.

By default, orientation is Qt::Horizontal and range is from 0 to 5.

The stars are rendered accoring to palette with the following color roles: ColorRole Element QPalette::Text outlines QPalette::Base unselected stars QPalette::Highlight selected stars
