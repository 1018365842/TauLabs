Detailed Description

The QxtProgressLabel widget is a label showing progress related time values.

QxtGui QxtProgressLabel is a label widget able to show elapsed and remaining time of a progress. Usage is as simple as connecting signal QProgressBar::valueChanged() to slot QxtProgressLabel::setValue().

Example usage:

QProgressBar* bar = new QProgressBar(this);
QxtProgressLabel* label = new QxtProgressLabel(this);
connect(bar, SIGNAL(valueChanged(int)), label, SLOT(setValue(int)));
