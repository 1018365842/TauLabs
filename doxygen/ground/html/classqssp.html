int16_t qssp::ssp_SendProcess ( )

Runs the send process, checks for receipt of ack, timeouts and resends if needed.

Parameters
thisport= which port to use
Returns
SSP_TX_WAITING - waiting for a valid ACK to arrive
SSP_TX_TIMEOUT - failed to receive a valid ACK in the timeout period, after retrying.
SSP_TX_IDLE - not expecting a ACK packet (no current transmissions in progress)
SSP_TX_ACKED - valid ACK received before timeout period.
