Detailed Description

GLC_Viewport : OpenGL Viewport.

An GLC_Viewport define Viewport with these specification

  • Default GLC_Camera
  • Max distance of view
  • Min distance of view
  • Angle of view
  • Maximum zoom factor

GLC_Viewport::GLC_Viewport ( QGLWidget *  GLWidget)
