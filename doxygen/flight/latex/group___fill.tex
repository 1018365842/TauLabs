\subsection{Detailed Description}
Fills the destination vector with a constant value.


\begin{DoxyPre}    
        pDst[n] = value;   0 <= n < blockSize.    
\end{DoxyPre}


There are separate functions for floating point, Q31, Q15, and Q7 data types.
