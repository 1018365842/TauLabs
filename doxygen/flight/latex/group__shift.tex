\subsection{Detailed Description}
Shifts the elements of a fixed-\/point vector by a specified number of bits. There are separate functions for Q7, Q15, and Q31 data types. The underlying algorithm used is\-:


\begin{DoxyPre}        
    pDst[n] = pSrc[n] << shift,   0 <= n < blockSize.        
\end{DoxyPre}


If {\ttfamily shift} is positive then the elements of the vector are shifted to the left. If {\ttfamily shift} is negative then the elements of the vector are shifted to the right.
