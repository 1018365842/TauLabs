Enables or disables the window mode. Window mode for comparators makes use of two comparators\-: C\-O\-M\-P1 and C\-O\-M2, C\-O\-M\-P3 and C\-O\-M\-P4, C\-O\-M\-P5 and C\-O\-M\-P6. In window mode, C\-O\-M\-Px and C\-O\-M\-Px-\/1 (where x can be 2, 4 or 6) non inverting inputs are connected together and only C\-O\-M\-Px-\/1 non inverting input can be used. e.\-g When window mode enabled for C\-O\-M\-P4, C\-O\-M\-P3 non inverting input (P\-B14 or P\-D14) is to be used. 

\begin{DoxyNote}{Note}
If the C\-O\-M\-Px is locked, E\-N\-A\-B\-L\-E/\-D\-I\-S\-A\-B\-L\-E the window mode can't be performed. To unlock the configuration, perform a system reset. 
\end{DoxyNote}
