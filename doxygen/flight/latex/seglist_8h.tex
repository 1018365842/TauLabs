\subsection{Detailed Description}
Segmented List data structure. A seglist is a linked list of segments. A segment is an array of ptrs to objects (with a pointer to the next segment). Seglists are used to implement Lists and Dicts.

This implementation of Seglist is straight. That is, the next pointer in the final segment contains C\-\_\-\-N\-U\-L\-L.

This implementation of Seglist is dense. That is, there are no gaps in a segment. All entries point to an object, except entries that are beyond the index of the last item.
