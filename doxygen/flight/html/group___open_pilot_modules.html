Detailed Description

The OpenPilot Modules do the majority of the control in OpenPilot. The PipXtreme Module The PipXtreme Module is the equivalanet of the System Module for the PipXtreme modem. it starts all the other modules.

This is done through the PIOS "PIOS Hardware abstraction layer",

which then contains hardware specific implementations

(currently only STM32 supported)

The OpenPilot Modules do the majority of the control in OpenPilot. The System Module starts all the other modules that then take care of all the telemetry and control algorithms and such. This is done through the PIOS Hardware abstraction layer which then contains hardware specific implementations (currently only STM32 supported)
