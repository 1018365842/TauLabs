Detailed Description

Control the UAV attitude to StabilizationDesired.

The main control code which keeps the UAV at the attitude requested by StabilizationDesired. This is done by comparing against AttitudeActual to compute the error in roll pitch and yaw then then based on the mode and values in StabilizationSettings computing the desired outputs and placing them in ActuatorDesired.
