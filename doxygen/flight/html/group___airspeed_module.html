Detailed Description

Calculate airspeed from diverse sources and update Airspeed UAV Object.

Calculate airspeed as a function of the difference between sequential GPS velocity and attitude measurements.

Use GPS data to estimate airspeed.

Communicate with airspeed sensors and return values.
